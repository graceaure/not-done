Quick Setup Guide

1

Start XAMPP Services

Open XAMPP Control Panel and start:

  • Apache - Web server
  • MySQL - Database server
Make sure both services show "Running" status
2

Test Database Connection

Verify your database setup is working:

Run Connection Test

This will check if MySQL is accessible

3

Run Automatic Installer

Let the installer set up everything for you:

Run Installer Now

Creates database, tables, and admin account automatically

4

Login to Dashboard

Use the default admin credentials:

Username: admin
Password: admin123
Go to Login
